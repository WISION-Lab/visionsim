  

SPSIM
=================================
Welcome to Spsim's documentation! This package helps researchers and developers simulate passive single photon cameras for testing in various applications.
It uses Blender to render out scenes at high frame rates, and includes both CLI tools and a library to assist with transforming formats,
emulating types of cameras, etc.
